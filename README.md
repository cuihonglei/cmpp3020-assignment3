## Running the Project

### Java Version
1. Navigate to the `java` folder.  
2. Compile and run the program using:  
```bash
java Main.java
```

### C# Version
1. Navigate to the `csharp` folder.  
2. Compile and run the program using:  
```bash
dotnet run
```
